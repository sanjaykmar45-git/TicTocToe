# Tic Toc Toe Game 

This is a simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features 

- Two-player mode (X and O)
- Reset / Restart game option
- Responsive and clean UI

## Technologies used 

- HTML 
- CSS 
- JavaScript
